📦 كيف ترفع المنتجات:
────────────────────────
1. افتح admin/index.html
2. اضغط "المنتجات" ← "إضافة منتج"
3. أدخل الاسم والأسعار والوصف
4. ارفع الصور (سحب وإفلات)
5. اضغط "حفظ المنتج"

🏷️ كيف تضيف فئة:
────────────────────
1. اضغط "الفئات" ← "فئة جديدة"
2. أدخل الاسم والأيقونة والترتيب
3. اضغط "حفظ الفئة"

🖼️ كيف تضيف بنر للسلايدر:
──────────────────────────────
1. اضغط "البنرات والسلايدر" ← "بنر جديد"
2. أدخل العنوان والألوان والأيقونة
3. اضغط "حفظ البنر"

✅ ملاحظات مهمة:
─────────────────
- المتجر مرتبط بـ Firebase مالتك مباشرة
- الطلبات تذهب تلقائياً لقسم "جديد" في نظامك
- الصور تُرفع على Firebase Storage
- كل التغييرات تظهر فوراً في المتجر
